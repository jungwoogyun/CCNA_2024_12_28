1. Network 는 무엇이며 무엇을 목적으로 하는가?
- 데이터교환을 목적으로 여러통신장치들을 연결한 통신망
- 연결 / 공유

2. Network 의 종류는 크게 ooo/ooo 로 나눌 수 있다
- LAN / WAN

3. LAN 의 특징에 대해서 서술하세요
- 근거리 통신망
- 통신 장치간 연결거리가 비교적 가까운(최대 2km 내외) 로 연결된 통신망
- 고속통신을 목적

4. WAN의 특징에 대해서 서술하세요
- 원거리 통신망(광역 통신망)
- 통신 장치간 연결거리가 비교적 먼 거리로 연결된 통신망
- ISP 업체가 보유한 통신망
- 고유 AS number 를 보유

5. InterNet 이란
- 수많은 통신망종류(LAN,PAN,WAN,GEN,MAN..) 을 연결한 거대 통신망

6. 프로토콜은 무엇인가?
- 통신을 하기위해 사전에 약속한 규약/규칙
- 구문(Syntex), 의미(Meaning) 타이밍(Timing) 
- 통신을 하기 위한 프로그램
- 1대이상의 기계간의 데이터 전달위해 사용

7. 네트워킹 모델의 목적은 무엇인가?
- 통신망 구성시 사용되는 지침서
- 통신망 하자발생시 문제해결을 위한 지침서
- 이기종간의 통신을 위한 지침서(OSI , TCP/IP)


8. TCP/IP 모델과 OSI 7 모델의 계층을 적고 각 계층별 역할을 적으세요
